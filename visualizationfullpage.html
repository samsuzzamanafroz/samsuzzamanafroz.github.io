Gravitational Wave Events: Catalog & Visualizations

Data Visualizations

All plots are generated using real data from LIGO, Virgo, and KAGRA gravitational-wave detectors.

Gravitational Wave Detections Catalog

This visualization presents gravitational-wave detections from the LIGO–Virgo–KAGRA (LVK) observing runs between 2015 and 2024. Each event corresponds to a compact binary merger, plotted versus the time of observation (horizontal axis) and the component masses (vertical axis, in solar masses). For each detection, the two markers connected by a vertical line represent the individual masses of the binary components, illustrating the mass ratio of the system. Events are color-coded by their luminosity distance, highlighting sources ranging from nearby mergers to events at cosmological distances. The upper axis shows the cumulative detection number, emphasizing the rapid growth of the gravitational-wave catalog as detector sensitivity improved across observing runs.

Component Masses and Luminosity Distance of Compact Binary Mergers

Scatter plot showing individual gravitational-wave events in the component mass–luminosity distance parameter space, using detections from the GWTC-3 (circles) and GWTC-4 (crosses) catalogs. Vertical line segments connect the lighter and heavier component masses of each binary system. Marker colors represent the effective inspiral spin parameter.

Chirp Mass and Effective Spin of Compact Binary Mergers

Scatter plot showing individual gravitational-wave events in the chirp mass–effective inspiral spin parameter space, as reported in the GWTC-3 (circles) and GWTC-4 (crosses) catalogs. Marker colors represent the luminosity distance of each source. The distribution highlights population-level trends and the diversity of compact binary merger properties observed by the LIGO–Virgo–KAGRA network across multiple observing runs.

playgwtc: A Gravitational-Wave Event Plotter

playgwtc is a Python-based command-line tool developed by Devesh Giri for fetching, processing, and visualizing gravitational-wave event data from the Gravitational Wave Open Science Center (GWOSC). The tool enables rapid generation of time–frequency Q-transform plots and theoretical waveform visualizations using published event parameters, producing high-quality figures suitable for scientific analysis and presentations.
